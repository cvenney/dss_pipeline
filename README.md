# dss_pipeline
Coverage filtration and differential methylation analysis for whole-genome methylation data with DSS. 01 and 02 scripts from https://github.com/kylewellband/methylUtil.

Dependencies:
- bedtools
- R v. 4.0.3
- data.table
- dplyr
